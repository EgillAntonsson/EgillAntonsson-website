My GitHub repository

These are scripts for REAPER DAW (Digital Audio Workstation). I store my scripts in a GitHub repository. You can find it here.

How to get my scripts into REAPER

The best way to to get the scripts is via the ReaPack REAPER extension, the package manager for REAPER made by cfillion. You could also download the scripts manually from the GitHub repository, but then you will need to update them manually as well. The ReaPack extension will automatically check for updates and install them for you and it is very easy to use, thus I recommend it.

  • Download and install ReaPack
  • In REAPER click menu item Extensions > ReaPack > Manage repositories...
  • Click Import/export... > Import repositories
  • Insert https://raw.githubusercontent.com/EgillAntonsson/REAPER-ReaScripts/refs/heads/master/index.xml and press OK
  • You should see Antonsson Scripts name in the list
  • Click Options... and checkmark install new packages when synchronizing and then click RePack menu item Synchronize packages

My scripts will be downloaded and installed in the Scripts folder in your REAPER installation folder. You can find it by clicking menu item Options > Show REAPER resource path in explorer/finder. The .lua scripts are in the Scripts subfolder, and the .jsfx scripts are in Effects subfolder.
